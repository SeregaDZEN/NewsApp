# NewsApp

Android-приложение для просмотра последних новостей с использованием [NewsAPI](https://newsapi.org/).

## 🚀 Технологии
- Kotlin
- Jetpack Compose
- Retrofit
- Coil (загрузка картинок)
- Coroutines
- GitHub Actions (CI/CD)

## 📱 Функции
- Загрузка новостей 
- Список новостей с заголовками, описаниями и картинками
- Красивый Material Design
